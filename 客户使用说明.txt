PiBot V3.28 Isolated Customer Guide
===================================

Required On The Customer Computer
---------------------------------
1. CH340 / USB serial driver
2. Chrome or Edge browser

Not Required
------------
- Python
- Arduino IDE
- XLoader
- Local proxy software

Web Page Steps
--------------
1. Connect the PiBot V3.28 Isolated board to the computer through USB.
2. Open the provided HTTPS website in Chrome or Edge.
3. The default homepage is English. For Chinese, click "中文" in the top-right language switch or open zh-cn.html.
4. Click "Connect Device".
5. The browser will show a serial port picker. Select the CH340 / COM port.
6. After the connection succeeds, firmware flashing, input/output testing, XYZ signal testing, and serial terminal commands can be used.

Flashing Notes
--------------
- Do not let Arduino IDE, XLoader, serial monitor tools, or another browser tab occupy the same COM port while flashing.
- If the page says the serial port cannot be opened, close other serial software, unplug and reconnect USB, then try again.
- After flashing completes, follow the page prompt and reconnect the serial port before testing.

Browser Requirements
--------------------
Web Serial requires a secure HTTPS website.

Recommended:
- Chrome
- Edge

Not recommended:
- Firefox
- Safari
- Mobile browsers

Driver Notes
------------
If no serial port appears in the browser picker, install the CH340/CH341 USB serial driver first, then unplug and reconnect the USB cable.

Official driver download:
https://www.wch-ic.com/downloads/CH341SER.ZIP.html

Local driver download:
Click "Local Download" in the CH340 Driver note on the web page, or open:

CH341SER.zip


中文辅助说明
============

客户电脑需要安装：
1. CH340 / USB 串口驱动
2. Chrome 或 Edge 浏览器

客户电脑不需要安装：
- Python
- Arduino IDE
- XLoader
- 本地代理程序

网页使用步骤：
1. 插入 PiBot V3.28 Isolated 主板 USB。
2. 使用 Chrome 或 Edge 打开商家提供的 HTTPS 网址。
3. 默认主页为英文版，如需中文可点击右上角“中文”或打开 zh-cn.html。
4. 点击网页上的 “Connect Device”。
5. 浏览器会弹出串口选择窗口，选择 CH340 / COM 端口。
6. 连接成功后，可以进行固件烧录、输入输出测试、XYZ 信号灯测试和串口命令操作。

烧录注意事项：
- 烧录时不要让 Arduino IDE、XLoader、串口调试助手占用同一个 COM 口。
- 如果提示串口打不开，请关闭其它串口软件，拔插 USB 后重试。
- 烧录完成后，按页面提示重新连接串口，再进行测试。

驱动说明：
如果网页连接时看不到串口，请先安装 CH340/CH341 串口驱动，然后重新插拔 USB。

官方下载地址：
https://www.wch-ic.com/downloads/CH341SER.ZIP.html

本地下载地址：
CH341SER.zip
